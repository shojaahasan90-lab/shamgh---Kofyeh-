How to deploy on Netlify:
1. Download the ZIP file.
2. Open Netlify.
3. Drag and drop the folder or ZIP into Netlify deploy.
4. If your old QR code already points to the same Netlify site, replace the current files with these and redeploy.

Important:
- If you want your exact real logo, replace logo.svg with your own logo file and keep the same name.
